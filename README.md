# مشروع صيدلية Codexe

هذا المشروع عبارة عن منصة إلكترونية متكاملة لإدارة الصيدلية، تم بناؤها باستخدام Next.js.

## التقنيات المستخدمة
- Next.js (App Router)
- Tailwind CSS
- TypeScript

## كيفية التشغيل
1. قم بعمل نسخة من المشروع: `git clone [رابط_المستودع]`
2. قم بتثبيت المكتبات: `npm install`
3. قم بتشغيل السيرفر: `npm run dev`
